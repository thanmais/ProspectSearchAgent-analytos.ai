# Assignment-analytos.ai
ProspectSearchAgent is an autonomous B2B lead discovery system powered by AI-assisted development (Cursor + Claude). It automates the process of identifying and enriching company and contact data that match a given Ideal Customer Profile (ICP) across multiple public and free-trial APIs.
**Hey! Here's ProspectSearchAgent - an AI-powered B2B prospect finder I built.**

To get started:
1. Open terminal in the `Leado` folder
2. Run: `python3 -m venv venv && source venv/bin/activate`
3. Run: `pip install -r requirements.txt`
4. Run: `python demo.py` (works instantly!)
5. Get a free Apollo key: https://www.apollo.io/
6. Add it to `.env` file
7. Run: `python main.py examples/icp_saas.yaml`
